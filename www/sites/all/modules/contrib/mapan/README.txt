
-- SUMMARY --

The Multi-admin pending approval notifications (MAPAN) module sends a mail 
to each user that have the 'administer users' permission. It is a plug-and-play
module: enable it and you are good to go.

-- REQUIREMENTS --

None.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- NOTES --

When an administrator receives a pending approval request, a link to his user 
profile form is appended to the body to quickly access a the setting that
enables him to disable this kind of notifications.
